# Czc's Blog

**克隆仓库地址**

```
git clone https://github.com/czc13611858691/czc13611858691.gihub.io.git
```

**安装nodejs**

[官网地址](https://nodejs.org/en/download/)

检查安装成功cmd指令

```
node -v
npm -v
```

**进入博客文件夹**

安装hexo-cli

```
npm install -g hexo-cli
```

```
hexo init blog
```

```
cd blog
```

```
hexo g
hexo s
```

浏览器输入

```
localhost:4000
```

**发布文章**

```
hexo new "文章名"
hexo g -d
```
